# Legacy SDXL MCIC Archive

This directory contains the original SDXL-oriented `mcic/` package, old configs, scripts, docs, and tests for the SDXL experiments.

They are archived and are not used by the FLUX Phase 1 paired warmup / B2 baseline path. New MA-RA-CDT Phase 1 work should use the top-level FLUX scripts (`build_cache.py`, `train_paired.py`, `eval_watcher.py`, `eval_b2.py`) and `configs/warmup.yaml`.
